# Analisando a precipitação na Amazônia Internacional

Este repositório é resultado de um deafio que foi proposto no [Laboratório de Pesquisa de Engenharia de Software](https://github.com/LAPES-Engenharia-de-Software)

## O que você vai encotrar neste repositório:

1. Um pequeno estudo da precipitação na amazônia internacional, ao longo de 21 anos.
2. Um arquivo .ipynb com os detalhes de como fiz este estudo.
3. Um arquivo .csv com os dados retirado da WorldClim que foi usado como base do estudo.

## Sobre a WorldClim:

WorldClim é um conjunto de dados climáticos globais que oferece informações de temperatura e precipitação em alta resolução espacial. Foi escolhida essa base de dados pela relevância na sua capacidade de fornecer dados climáticos detalhados para quase todas as regiões do mundo.

Para mais informações, visite o [site oficial do WorldClim](http://worldclim.org/).
